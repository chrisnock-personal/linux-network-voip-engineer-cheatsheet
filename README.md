## What's Covered

**Networking**
- OSI model, Layer 2 & Layer 3 fundamentals
- IP addressing & subnetting, VLANs, NAT
- TCP/UDP, DNS, DHCP, routing (static, OSPF, BGP)
- Cisco IOS commands, ACLs & firewall rules
- QoS, port mirroring/SPAN, IDS & IPS
- Network troubleshooting methodology

**Voice — VoIP, UC & Trading**
- VoIP fundamentals, codecs, and quality metrics (MOS, jitter, packet loss)
- SIP call flow (INVITE, TRYING, RINGING, 200 OK, ACK, BYE)
- RTP/RTCP, SRTP, NAT traversal, SBCs
- VoIP troubleshooting (one-way audio, call drops, echo)
- Unified Communications — Microsoft Teams Direct Routing, Cisco CUCM, Avaya Aura
- Trading voice — turrets, IPC Unigy, hoot-and-holler, compliance recording

**Monitoring & Observability**
- The three pillars: metrics, logs, traces
- SNMP, Syslog, NetFlow, synthetic monitoring
- Common monitoring platforms (Grafana, Prometheus, Splunk, Zabbix, SolarWinds)
- NOC operations — incident severity, P1 handling, SLAs, MTTR/MTBF
- Incident management, blameless post-mortems, 5 Whys, change management

**Linux & DevOps**
- Linux command line essentials — networking, process, log, and file management
- Linux networking — bonding, bridging, iptables/nftables, MTU issues
- Linux virtualisation — KVM/QEMU, LXC, Vagrant
- Linux filesystem management — LVM, disk usage, common faults
- Linux user management — permissions, sudo, SSH keys, PAM
- Linux monitoring — htop, iostat, journalctl, dmesg, monitoring agents
- Docker & Kubernetes — core concepts and essential commands
- Python basics for networking — netmiko, paramiko, requests, log parsing
